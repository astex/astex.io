  • needs.py Sept 2014‑Dec 2014

    Needs.py is a small, open source (MIT License) library for manipulating boolean conditionals in a convenient way. It allows the programmer to perform boolean logic on Need instances storing the resulting instance, but not executing the conditionals until needed. This is very useful for permission management in a web framework in which case permissions can be both complicated and database-intensive.
